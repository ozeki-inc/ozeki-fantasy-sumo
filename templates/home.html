This is a very simple open source fantasy sumo league engine. Users can create a league by selecting scoring rules, number of players, which tournament they would like to play, etc. Once the league is created and rosters are entered, the server will automatically fetch match results and update league scores. No accounts are needed, if you would like you can provide a public key upon league creation that you can prove ownership of a roster. For the moment, once a team is created no changes to the roster are possible. This may change in future releases.

You can view the source code, find hosting instructions, and contribute on the Official Ozeki Inc. Github.
