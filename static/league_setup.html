League settings explained:

  • Bansho: select the tournament you'd like to play.
  • Start day: which day of the tournament the league should start (1-15)
  • How many teams: number of players in the league
  • Roster size: number of wrestlers each team will draft.
  • Win pts: points to be awarded for a match win
  • Lose pts: points to be removed for a match loss
  • Rank bonus: Percentage (0-1) added or removed according to difference in wrestler ranks. (e.g. with a bonus value of 0.1, a Maegashira beating a Yokozuna will multiply the win points by (1 + bonus * 4) since a Yokozuna is 4 ranks higher.)
  • Rival multiplier: points awarded in matches where both wrestlers belong to the league's rosters will be multiplied by the given factor.
